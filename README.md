# Hotel-Reservation-System

## Technologies used
- Java Swing
- JDBC
- MySQL

## Description
This is a desktop-based application developed using Java Swing that allows users to manage hotel room bookings efficiently.

## Features

- Add Customer Details
- Room Booking System
- Employee Management
- Database Connectivity using JDBC

## Requirements
- MySQL installed
- JDBC Driver (included in the lib folder)
- rs2xml.jar - used to display database ResultSet data in JTable.
